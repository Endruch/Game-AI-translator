БЫСТРЫЙ СТАРТ

1. Добавьте API ключ Claude в Settings (⚙)
2. Наведите зелёную рамку на текст в игре
3. Нажмите Shift+F9 для перевода

═══════════════════════════════════════════════════════════

ГОРЯЧИЕ КЛАВИШИ

Shift+F9  - Перевести текст в области захвата
Shift+F8  - Показать/скрыть зелёную рамку

Вы можете изменить горячие клавиши в Settings → General

═══════════════════════════════════════════════════════════

ЦВЕТОВЫЕ ФИЛЬТРЫ

Слева в окне переводчика находятся цветовые фильтры.
Они позволяют выбрать только определённые цвета текста.

• Когда фильтрация ВКЛЮЧЕНА (Settings → Overlay):
  Распознаётся только текст выбранных цветов

• Когда фильтрация ВЫКЛЮЧЕНА:
  Распознаётся весь текст, независимо от цвета

Используйте фильтры, чтобы игнорировать UI элементы
и захватывать только игровой текст/чат.

ЗЕЛЁНАЯ РАМКА (OVERLAY)

• Перетаскивайте рамку на нужное место
• Изменяйте размер, потянув за правый нижний угол
• Кнопка замка 🔒 блокирует положение и размер
• Настройте цвет и толщину границы в Settings → Overlay

═══════════════════════════════════════════════════════════

АВТО-ПЕРЕВОД

Settings → Auto-Translate

Автоматически проверяет область захвата каждые N секунд.
Перевод выполняется только если текст изменился.

Полезно для автоматического перевода диалогов в играх.

═══════════════════════════════════════════════════════════

ИСТОРИЯ ПЕРЕВОДОВ

Settings → History

Сохраняет все переводы в текстовые файлы:
translation_log_YYYY-MM-DD.txt

Один файл на день. Можно выбрать свою папку для сохранения.

НАСТРОЙКИ API КЛЮЧА

1. Перейдите на: console.anthropic.com
2. Создайте API ключ
3. Скопируйте ключ (начинается с sk-ant-)
4. Вставьте в Settings → General → Claude API Key

ВАЖНО: Храните ключ в секрете!

СИСТЕМНЫЙ ТРЕЙ

Программа сворачивается в трей (нижний правый угол).

ПКМ по иконке в трее:

• Settings - открыть настройки

Двойной клик по иконке - показать окна

РЕШЕНИЕ ПРОБЛЕМ

❌ "No text recognized"
   → Выключите цветовые фильтры в Settings → Overlay
   → Убедитесь что рамка покрывает весь текст
   → Проверьте что текст достаточно крупный

❌ "API key is missing"
   → Добавьте ключ в Settings → General

❌ Перевод не работает
   → Проверьте API ключ
   → Проверьте интернет соединение
   → Убедитесь что не превышен лимит API

❌ Горячие клавиши не работают
   → Перезапустите программу с правами администратора
   → Проверьте что клавиши не заняты другой программой
